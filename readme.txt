TestAutomation project

Content of the project:
constants package - used constants;
elements package - classes with WebElements;
logic package - classes with test methods;
util package - classes - helpers;
tests package - test class


Test running
For test running use commands: mvn test (from folder of project)

mvn test running testng.xml which located in test/resources -> testng.xml
testNG reports will be create in target/surefire-reports -> index.html
